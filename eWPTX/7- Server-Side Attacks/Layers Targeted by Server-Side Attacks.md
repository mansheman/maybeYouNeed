# Layers Targeted by Server-Side Attacks

In a typical architecture, server-side attacks usually target:

- **Web server layer**: HTTP gateway, static assets, reverse proxy
- **Application server layer**: dynamic processing + business logic
- **Data access layer** (indirectly): impacted via app/web server compromise

Client-side attacks target the browser/presentation layer.
