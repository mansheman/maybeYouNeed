# Web Server vs Application Server

|Aspect|Web Server|Application Server|
|---|---|---|
|Role|Serve static content, route requests|Execute business logic, dynamic responses|
|Interaction|Client via HTTP/HTTPS|DB/APIs/internal services|
|Processing|Lightweight|Heavy processing|
|Output|Files (HTML/CSS/JS)|Computed responses (HTML/JSON)|
|Examples|Nginx, Apache, IIS|Node.js, Django, Spring Boot|
