# What is Web Application Architecture?

Web application architecture refers to the structure/framework that defines how components interact to deliver the application.

It includes:

- front-end (presentation)
- back-end (business/app layer)
- data storage
- communication between components

A solid architecture influences:

- functionality
- performance
- scalability
- adaptability
