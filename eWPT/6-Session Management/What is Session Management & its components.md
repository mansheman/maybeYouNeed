# What is Session Management & its components
# 

## Overview
- Process of securely handling and maintaining user sessions.  
- A session begins when a user logs in and ends when they log out or it expires.  
- The application must recognize the user, store data, and manage access.  
- Ensures security, user experience, and state management.  

## Components
- **Session Identifier**: Unique token (session ID) that links user requests to session data.  
- **Session Data**: Stored on the server (authentication status, preferences, temporary info).  
- **Session Cookies**: Stored in the browser; contain session ID to maintain state.
