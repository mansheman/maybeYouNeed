# Web Enumeration & Information Gathering
* Information gathering is the first step in any penetration test
* the more information you have on the target the more successful you will be
*  information gathering is two types
	*  Passive information gathering: gathering as much information as possible without actively engaging with the target
	* Active information gathering : gathering as much information as possible by actively engaging with the target (you require authorization )
	
* ## Passive information Gathering
	*  Identifying domain names and domain ownership information
	-  Discovering hidden/disallowed files and directories 
	-  Identifying  web server IP address and DNS records
	-  Identifying web technologies used on the website
	-  WAF detection
	-  Identifying subdomains
	-  Identify website
- ## Active Information Gathering
	-  Downloading & analyzing web app source code
	-  Port scanning & service discovery
	-  Web server fingerprinting
	-  Web application scanning
	-  DNS Zone transfer
	-  Subdomain enumeration via Brute Force
