# Web Proxies

A web proxy (interception proxy) captures, analyzes, and modifies HTTP/HTTPS requests between a client and a server.

Used in web application pentesting to intercept traffic, analyze behavior, and manipulate requests or responses to uncover vulnerabilities.

Essential for manual testing of web applications.

Common tools: Burp Suite, OWASP ZAP.
