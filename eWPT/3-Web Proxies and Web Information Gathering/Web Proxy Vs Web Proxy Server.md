# Web Proxy vs Web Proxy Server

## Web Proxy

- Intercepts, analyzes, and modifies HTTP/HTTPS requests between client and server.
    
- Used in security testing and web application assessments.
    
- Example tools:
    
    - Burp Suite
        
    - OWASP ZAP
        

## Web Proxy Server

- Proxies general internet traffic.
    
- Used for traffic filtering, caching, and bandwidth optimization.
    
- Example tools:
    
    - Squid Proxy
        

Illustrations clarify the distinction between interception proxies (security) and proxy servers (networking).
