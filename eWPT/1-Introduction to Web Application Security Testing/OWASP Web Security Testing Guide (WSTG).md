# OWASP WSTG Checklist

[[OWSAP checklist]] : A **spreadsheet-based checklist** designed to help track completed and pending test cases for web application security assessments.

###### Key Aspects:
*   **Foundation:** Based on the **OWASP Web Security Testing Guide (WSTG)**, providing a practical implementation of its methodology.
*  **Purpose:**
    *   Facilitates a comprehensive penetration testing framework for web app pentesters.
    *   Outlines detailed, granular tests for common web misconfigurations, flaws, and vulnerabilities.
*   **Includes:**
    *   OWASP Risk Assessment Calculator.
    *   Summary Findings template.
*   **Benefit:** Serves as a practical, hands-on tool for standardizing and executing web application security testing.
