# XSS Types
**XSS Categories and Objectives**

**Main Types of XSS Attacks:**

- **Stored (Persistent):** Malicious script is permanently stored on the target server (e.g., in a database) and served to users.
    
- **Reflected:** Malicious script is reflected off a web server (e.g., in an error message or search result) and sent to the victim via a crafted link.
    

**Common Objectives of XSS Attacks:**

- **Cookie Stealing / Session Hijacking:** Capture session cookies to impersonate users.
    
- **Browser Exploitation:** Leverage browser vulnerabilities to execute further attacks.
    
- **Keylogging:** Monitor and record user keystrokes.
    
- **Phishing:** Inject fake login forms to steal user credentials.
    
- **Others:** Various forms of data theft, redirection, and user manipulation.
