# What is XSS
### **Cross-Site Scripting (XSS)**

**Definition:**  
XSS is a client-side web vulnerability that allows attackers to inject malicious scripts into web pages viewed by other users.

**Cause:**

- Lack of input sanitization or validation in web applications.
    

**How It Works:**

- Attackers inject malicious code into a vulnerable web application.
    
- Since the vulnerability is on the client side, the script executes in the victim’s browser.
    

**Affected Technologies:**

- JavaScript
    
- Flash
    
- CSS
    
- Other client-side scripting languages
    

**Impact:**

- Executed scripts can steal data, hijack sessions, redirect users, or deface websites.
