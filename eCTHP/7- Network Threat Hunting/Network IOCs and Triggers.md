# Network IOCs and Triggers

Network threat hunting starts with a network-based hypothesis or IOC rather than primarily with ATT&CK TTPs.

Typical triggers:

- intelligence
- SOC or network-team reports
- monitoring alerts
- live packet capture
- previously captured traffic

Common network IOCs:

- IP addresses and ranges
- domain names and suspicious subdomains
- lookalike domains
- protocol use on non-typical ports
- traffic that violates normal protocol standards
