# Capturing Network Traffic

Where traffic is captured determines what you can actually see.

Common options:

- network tap
- port mirroring or SPAN
- endpoint capture

Capture helps reveal:

- command and control activity
- payload downloads
- lateral movement
- exfiltration attempts
