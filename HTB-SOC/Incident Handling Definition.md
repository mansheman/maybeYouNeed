# Incident Handling Definition
[[Tags/SOC]] [[Tags/HTB]] [[Tags/IncidentHandling]]

Incident handling (IH) is a defensive capability against cybercrime. It is a necessity for organizations to protect data confidentiality, integrity, or availability.

**Definition:**
> Incident handling is a clearly defined set of procedures for managing and responding to security incidents in a computer or network environment.

It is not limited to intrusion incidents; it also covers malicious insiders, availability issues, and loss of intellectual property.
