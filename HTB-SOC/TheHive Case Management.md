# TheHive Case Management
[[Tags/SOC]] [[Tags/HTB]] [[Tags/CaseManagement]]

TheHive is a case management platform for cybersecurity teams to efficiently handle incidents by processing alerts.

### Key Features
- **Centralized Hub**: Collects alerts from various devices (e.g., Wazuh).
- **MITRE ATT&CK Integration**: Imports TTPs to enrich incident analysis.
- **Alert to Case**: Link multiple relevant alerts within a single case.
