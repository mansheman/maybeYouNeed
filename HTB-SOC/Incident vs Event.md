# Incident vs Event
[[Tags/SOC]] [[Tags/HTB]] [[Tags/IncidentHandling]]

Organizations must distinguish between normal system activity and security-relevant occurrences.

### Event
An **event** is an action occurring in a system or network.
- Examples: Sending an email, mouse click, firewall allowing a connection.

### Incident
An **incident** is an event with a **negative consequence**.
- Examples: System crash, unauthorized access to sensitive data, natural disasters.

**IT Security Incident Definition:**
An event with a **clear intent to cause harm** performed against a computer system.
- Examples: Data theft, funds theft, installation of malware.
